## Number Guessing Game

A simple command-line game written in Python where the user tries to guess a randomly generated number.

### Features
- Custom number range chosen by the user
- Attempt counter
- Maximum attempt limit
- Input validation

### How the Game Works
1. The user enters a lower bound and an upper bound.
2. The program generates a random number within that range.
3. The user tries to guess the number.
4. The program tells whether the guess is too high or too low.
5. The game ends when the user guesses correctly or runs out of attempts.
